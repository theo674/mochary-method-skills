# Mochary Method Skills

AI agent skills based on [Matt Mochary's](https://docs.google.com/document/d/18FiJbYn53fTtPmphfdCKT2TMWH-8Y2L-MLqDk-MFV4s/edit?tab=t.0) coaching frameworks. Built for [OpenClaw](https://github.com/openclaw/openclaw) and compatible AI agent platforms.

## Skills

| Skill | Description |
|---|---|
| **[make-people-feel-heard](make-people-feel-heard/)** | The "Heard" technique for making people feel heard during feedback, conflict, or painful conversations |
| **[fear-and-anger-give-bad-advice](fear-and-anger-give-bad-advice/)** | Recognizing and managing decisions made from fear or anger |
| **[decision-making](decision-making/)** | 7 frameworks: separate decision/implementation, buy-in methods, IPS, Type 1/2, RAPID, "I intend to...", loudest voice |
| **[accountability](accountability/)** | 8 frameworks: agreement trackers, DRIs, clean escalation, first 2 minutes, AORs, OKRs, accountability partners, goals |
| **[ceo-role](ceo-role/)** | Mission/vision/values, Top Goal, CEO role definition, Zone of Genius, Exec Ops (Chief of Staff) |
| **[goals](goals/)** | Goal creation, input-based actions, bias to action (first 2 minutes), structured weekly reviews, coaching anti-patterns |

## Structure

Each skill follows the [AgentSkills spec](https://docs.openclaw.ai):

```
skill-name/
├── SKILL.md              # Instructions + quick reference
└── references/           # Full method detail
    └── <method>.md
```

## Sources

All content is derived from publicly available Mochary Method documents and videos by Matt Mochary. Links to original Google Docs and Loom/YouTube videos are included in each reference file.
